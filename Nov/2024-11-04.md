### 💡 새로 배운 내용

#### 1. 함수 선언문 vs 함수 표현식

**함수 선언문** <br />

```
const examFunction = () => {
  // 함수 내용
};
```

- 함수 자체를 **선언**하는 방식
- 호이스팅 O
- 이름이 명확하게 드러나므로 가동성 🆙

**함수 표현식** <br />

```
function examFunction() {
  // 함수 내용
}
```

- 화살표 함수 또는 익명 함수를 변수에 할당하는 방식
- 호이스팅 X (함수 선언 전에 사용할 수 없음)
